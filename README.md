# 📌 Sidebar Menu — CodingLab

Um menu lateral (sidebar) completo com logo, links de navegação com ícones, ícones de redes sociais e botão de fechar — desenvolvido com HTML, CSS e JavaScript.

---

## 📸 Preview

> Sidebar branca aberta no lado esquerdo da tela, com logo **CodingLab** e botão ✕ no topo. Lista de 8 links com ícones individuais e 4 ícones de redes sociais na base. O restante da página exibe fundo com textura de ondas multicoloridas (rosa, azul, verde).

---

## 🚀 Tecnologias

- **HTML5**
- **CSS3**
- **Font Awesome** (ícones dos links e redes sociais)

---

## 📁 Estrutura do Projeto

```
sidebar-menu/
├── index.html
├── style.css
```

---

## ⚙️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/sidebar-menu.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd sidebar-menu
   ```

3. Abra o `index.html` no navegador ou via **Live Server** no VS Code:
   - Acesse em: `http://127.0.0.1:5500`

---

## 🎯 Funcionalidades

- Sidebar com abertura/fechamento animado
- Botão **✕** para fechar o menu
- Logo **CodingLab** no topo da sidebar
- 8 links de navegação com ícones
- 4 ícones de redes sociais na base
- Fundo principal com textura animada de ondas coloridas
